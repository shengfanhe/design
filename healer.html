Project Info

As a company known as "APP factory", Bytedance has a super complicated data orgnization which centralize the data resources as a platform. The data dependency is one of the most critical part of its data governance. This product is to build a data SLA feature to make the data dependencies easier to manage by aligning the data engineering expectations between different orgs and business lines.

What is data SLA?

It’s a public promise to deliver a quantifiable level of service. Data SLAs reduce disagreement and create clarity among engineering orgs.

From a project perspective, users are able to get a holistic view of all tasks layed out in the SLA timeline. See the comparison between the expected and actual output time.

From a task perspective, users can browse all projects that are dependent on this task and get an intuitive view of the SLA time of the task comparing with the upstream and downstream tasks in each project.
