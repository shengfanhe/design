Magnify post-sale orchestration system

Project Info

Magnify started with a simple idea of putting customer success on autopilot. This is realistic by using the data from customer interacting with the product or service. I joined the startup team since the team was only CEO and CTO. I worked closely with them and also the specialists from the VC. We shipped the MVP in 6 months and take another 6 months to polish the product and acquired 2 customers.

Customer Success Automated.

Use the integrated data sources to build and segment and define the flow logic in a drag-n-drop user interface.

Dynamic input feature fully exploite the power of data, which takes the automation capabilities to the next level.

Set a goal for the automated journey and user can create an experiment for the 2 versions of journey to test out which one works better for the goal.

With data science and predictive AI technology, we can inform the customer success managers about the churn factors that need their attention. Then action can be taken into an automation workflow.
